### MyWays Challange
---
This is Code for MyWays React Native Challange



## Installation

1. Clone the repo

   ```sh
   git clone https://github.com/aryan-dwivedi/MyWays.git
   ```

2. Install NPM packages

   ```sh
   yarn install
   ```
   
3. Install POD files for ios

   ```sh
   npx pod install ios
   ```

4. Start the simulator

   ```sh
   yarn ios
   ```
